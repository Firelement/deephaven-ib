writeParquetTables(sources, tableDefinition, writeInstructions, destinations, groupingColumns)[source]#
Writes tables to disk in parquet format to a supplied set of destinations. If you specify grouping columns, there

must already be grouping information for those columns in the sources. This can be accomplished with .groupBy(<grouping columns>).ungroup() or .sort(<grouping column>).

Parameters

  • sources – (io.deephaven.engine.table.Table[]) - The tables to write

  • tableDefinition – (io.deephaven.engine.table.TableDefinition) - The common schema for all the tables to write

  • writeInstructions – (io.deephaven.parquet.table.ParquetInstructions) - Write instructions for customizations while writing

  • destinations – (java.io.File[]) - The destinations paths. Any non existing directories in the paths provided are created. If there is an error any intermediate directories previously created are removed; note this makes this method unsafe for concurrent use

  • groupingColumns – (java.lang.String[]) - List of columns the tables are grouped by (the write operation will store the grouping info)
